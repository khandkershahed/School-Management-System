# School Management System

A complete School Management System built with Laravel to simplify administration, improve communication, and streamline academic operations.  
This platform helps schools manage **students, teachers, parents, classrooms, grades, attendance, exams, online classes, fees, invoices, receipts, audit logs**, and more â€” from one place.

**Developer:** Khandker Shahed  
**Company:** Codershat

---

## Features

- Authentication (Laravel Jetstream)
- Role & Permission Management (Admin / Teacher / Parent / Student)
- Grades & Classrooms Management
- Students, Teachers, Parents Management
- Attendance Management + Export Reports
- Discussions (Posts & Comments)
- Fees Management (Fee Types, Study Fees)
- Student Invoices, Receipts, Payment Vouchers, Processing Fees
- Exams & Subjects
- Online Classes (Meetings)
- Academic Years
- Audit Logs (System operations tracking)
- Multi-language Support (English + Bangla)

---

## Tech Stack

- Laravel (PHP 8.2+)
- Laravel Jetstream (Auth)
- MySQL / MariaDB
- Tailwind CSS
- Mcamara Laravel Localization (Multi-language)
- Spatie Laravel Permission (Roles & Permissions)

---

## Requirements

Make sure your server has:

- PHP **8.2+**
- Composer
- MySQL / MariaDB
- Node.js + NPM (for frontend assets)
- Enabled PHP extensions (common Laravel requirements)
